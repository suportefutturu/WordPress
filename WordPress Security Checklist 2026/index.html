WordPress Security Checklist

Proteja sua instalação com ações práticas e ferramentas atualizadas.

Importante: A maioria das invasões ocorre por plugins desatualizados ou senhas fracas, não por falhas no núcleo do WordPress. Use este checklist para reduzir riscos de forma eficiente.

0 de 30 itens concluídos

wp-config.php

2 itens
  • Chaves de segurança (Salts) Gerador oficial
    Atualize as chaves usando o Gerador do WordPress.org. Isso invalida cookies antigos.
  • Desabilitar editor de arquivos
    define('DISALLOW_FILE_EDIT', true); no wp-config.php.

Página de Login

9 itens
  • Bloqueio de força bruta Lockspire / Solid Security
    Limite tentativas de login para evitar ataques automatizados.
  • Autenticação de 2 fatores (2FA) Duo for WordPress / WP 2FA
    Adicione uma camada extra com TOTP ou push notification.
  • Login com e-mail
    Use e-mail em vez de username. (Solid Security ou função em functions.php).
  • Alterar URL de login WPS Hide Login / Lockdown WP Admin
    Troque /wp-login.php por um caminho personalizado.
  • Senhas fortes obrigatórias
    Mínimo 12 caracteres, com maiúsculas, números e símbolos.
  • Mensagens de erro genéricas
    Evite revelar se o usuário existe.
  • Remover links para login
    Cheque o tema para links óbvios para /wp-admin.
  • Rotatividade de senha
    Altere senhas administrativas a cada 90 dias.
  • Desabilitar API REST (se não usada) Disable WP REST API
    Bloqueie acesso não autenticado à API REST.

Painel Administrativo

5 itens
  • Proteção extra na pasta wp-admin
    Autenticação HTTP (.htpasswd).
  • WordPress sempre atualizado
    Mantenha núcleo na última versão.
  • Sem usuário "admin"
    Crie conta com nome único e exclua "admin".
  • Use conta Editor para publicar
    Privilégio mínimo para tarefas diárias.
  • Monitoramento de integridade Wordfence / Lumiverse
    Alerta sobre alterações suspeitas em arquivos.

Tema

4 itens
  • Atualizar tema ativo
    Mantenha o tema e child-theme atualizados.
  • Apagar temas inativos
    Exclua, não apenas desative.
  • Fontes confiáveis
    Instale apenas do repositório oficial ou desenvolvedores renomados.
  • Remover versão do WP no tema Meta Generator Remover
    Oculte a tag generator do header.

Plugins

4 itens
  • Atualizar todos os plugins
    Ative atualizações automáticas para segurança.
  • Excluir plugins inativos
    Remova completamente.
  • Fontes confiáveis
    Nunca use plugins "nulled".
  • Substituir plugins abandonados
    Se sem atualização há >9 meses, busque alternativa.

Banco de Dados

3 itens
  • Alterar prefixo das tabelas Change Table Prefix
    Troque 'wp_' por algo único.
  • Backups automáticos semanais JetBackup / UpdraftPlus
    Backup off-site (Drive, S3).
  • Senha forte do usuário do banco
    Use senha complexa exclusiva para o MySQL.

Hospedagem

3 itens
  • Acesso apenas via SFTP/SSH
    Nunca use FTP simples.
  • Permissões corretas: pastas 755 / arquivos 644
    wp-config.php com 640.
  • Bloqueio de navegação de diretórios e arquivos sensíveis
    Options All -Indexes e bloqueio de readme.html.
